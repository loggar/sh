# bash operators

| Bash Operator | Operator | Description           |
| ------------- | -------- | --------------------- |
| -eq           | ==       | Equal                 |
| -ne           | !=       | Not equal             |
| -gt           | >        | Greater than          |
| -ge           | >=       | Greater than or equal |
| -lt           | <        | Less than             |
| -le           | <=       | Less than or equal    |
| -z            | == null  | Is null               |
